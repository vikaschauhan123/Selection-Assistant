Selection Assistant — User Guide

Select text on any webpage to hear it, rephrase it, translate it, or define it.

1. Settings

Click the Selection Assistant icon in your toolbar to open settings.

Enable toolbar on selection

A single switch at the top turns the whole extension on or off. Flipping it takes effect immediately on every open tab — no page refresh needed. Turn it off any time the toolbar is getting in your way (e.g. while working in a text-heavy editor).

API key (for Rephrase)

Read Aloud, Translate, and Define all work with no setup. Only Rephrase needs an API key, since it calls an AI model.

  1. Paste an API key into the API Key field — the provider is auto-detected from the key's prefix:
Key prefix Provider Get a key
sk-ant-... Anthropic (Claude) console.anthropic.com/settings/keys
sk-... OpenAI platform.openai.com/api-keys

A hint below the field confirms which provider was detected, or shows an error if the key doesn't start with either prefix.

Target language (for Translate)

Type to search — the list narrows as you type and shows up to 10 matches at a time, scrollable for more. It includes 50+ languages, with a range of Indian regional languages (Hindi, Bengali, Tamil, Telugu, Marathi, Gujarati, Punjabi, Urdu, Kannada, Malayalam, Odia, Assamese, Nepali, Sanskrit, Sindhi, Konkani, Maithili, Bhojpuri, Manipuri, Dogri, and more). Click a result to select it, then click Save.

Your API key is stored only in your own browser's chrome.storage.sync and is sent directly from your browser to OpenAI or Anthropic — it never passes through any other server.

2. Using it on a page

Select any text on a webpage. A small toolbar appears just below your selection. Hover over any button to see a tooltip explaining what it does.

🔊 Read Aloud

Speaks the selected text using your browser's built-in voice. Click again while it's speaking to stop. Use the ⏱ 1x button next to it to cycle playback speed: 0.75x → 1x → 1.25x → 1.5x → 2x — changing the speed mid-speech restarts playback at the new rate.

📋 Copy

Copies the raw selected text straight to your clipboard — no need to run Rephrase or Translate first if you just want the original text copied. The icon briefly turns into a ✅ to confirm.

✨ Rephrase

Sends the selection to your configured AI provider and shows a rewritten version in the result panel. Requires an API key (see above).

🌐 Translate

Translates the selection into your configured target language using Google Translate's free endpoint. No API key needed.

📖 Define

Only appears when you select a single word. Looks up its part of speech and definitions via a free dictionary API.

Moving and minimizing the toolbar

  • Grab the ✛ handle at the start of the toolbar to drag it anywhere on screen — the cursor changes to a 4-way move icon.
  • Click − at the end of the toolbar to shrink it down to a small ⋯ dot; click that dot to expand it again.
  • Both where you dragged it to and whether it's minimized are remembered — your next text selection on the same page opens the toolbar in that same spot and state. This resets when you refresh the page.

The result panel

  • Click the 📋 icon (top-right) to copy the result to your clipboard.
  • Click × to close it, click anywhere outside it, or press Esc.
  • The toolbar and panel automatically stay on-screen — they flip above/below and left/right of your selection depending on where there's room.
  • Both automatically invert their color scheme to stay visible against the page: a light toolbar/panel on dark sites, a dark one on light sites.

3. Troubleshooting

"Add your API key in the extension's settings popup first"

Shown when you click Rephrase with no key saved. Open the popup, add a key, and save.

"Your OpenAI/Anthropic key has run out of credits/rate limit"

Your account needs more credits, or you've hit a rate limit. Check your provider's billing dashboard.

"This extension was updated or reloaded..."

This appears on tabs that were already open when you reloaded the extension in chrome://extensions during development. Simply refresh that tab.

If the toolbar doesn't appear at all on a specific site (e.g. an app with heavy custom text editors), that site may render your selection inside an iframe or block standard selection events — this can require site-specific handling beyond what a generic content script covers.

4. Privacy

  • Rephrase sends the selected text to OpenAI or Anthropic, per your saved key.
  • Translate sends the selected text to Google Translate's public endpoint.
  • Define sends the selected word to api.dictionaryapi.dev.
  • Read Aloud never leaves your browser — it uses the local Web Speech API.
  • No data is collected, logged, or stored by the extension itself beyond your saved settings.
